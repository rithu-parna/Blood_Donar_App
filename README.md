# 🩸 Lifeline | Blood Donation Management System

Lifeline is a sophisticated, high-performance blood management platform designed to bridge the gap between donors and those in need. Built with a modern tech stack, it provides a seamless experience for both administrators and donors, ensuring that life-saving resources are managed efficiently and transparently.

![Lifeline Banner](https://images.unsplash.com/photo-1615461066870-607753630236?q=80&w=2000&auto=format&fit=crop)

## ✨ Features

### 🏢 Advanced Admin Dashboard
- **Real-time Analytics**: High-fidelity charts showing donation trends and request patterns using `Chart.js`.
- **Inventory Management**: Dynamic tracking of blood units across all groups (A+, O-, etc.) with critical level alerts.
- **Hospital Integration**: Direct management of requests from partner hospitals like City General and Mercy Care.
- **Donor Directory**: Comprehensive database of registered donors with status tracking and donation history.
- **Micro-interactions**: Powered by `Framer Motion` for a premium, responsive feel.

### 🏠 Customer Portal
- **Modern Landing Page**: Aesthetic and responsive interface built with `Material UI`.
- **Mission-Driven Design**: Focuses on community engagement and facilitating the donation process.
- **Fast Navigation**: Single-page application architecture for instant transitions.

### 🚀 Performance & UX
- **Glassmorphism UI**: Modern aesthetic with blurred backgrounds and sleek gradients.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile viewing.
- **Animated Components**: Smooth transitions and pulse effects for critical alerts.

## 🛠 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **UI Library**: [Material UI (MUI) v7](https://mui.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Charts**: [Chart.js](https://www.chartjs.org/) & [React-Chartjs-2](https://react-chartjs-2.js.org/)
- **Routing**: [React Router v6](https://reactrouter.com/)
- **Carousel**: [React Slick](https://react-slick.neostack.com/)

## 🏁 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/blood-donor-app.git
   cd blood-donor-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
